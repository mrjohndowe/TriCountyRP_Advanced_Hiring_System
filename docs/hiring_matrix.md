# Hiring Decision Matrix

Use this matrix after the interview scorecard is completed.

## Scoring Categories

| Category | Max Score |
|---|---:|
| Communication | 5 |
| Maturity / Professionalism | 5 |
| RP Knowledge | 5 |
| Department Knowledge | 5 |
| Scenario Judgment | 5 |
| Activity / Reliability | 5 |
| Leadership Potential *(optional)* | 5 |

## Standard Total

Without leadership category: **30**
With leadership category: **35**

## Decision Bands

### Standard Applicant (30-point scale)

| Score Range | Recommendation |
|---|---|
| 26–30 | Strong Accept |
| 21–25 | Accept |
| 16–20 | Probationary Accept |
| 0–15 | Deny |

### Leadership Applicant (35-point scale)

| Score Range | Recommendation |
|---|---|
| 30–35 | Strong Accept |
| 24–29 | Accept |
| 18–23 | Probationary Accept |
| 0–17 | Deny |

## Final Review Factors

Before making a final decision, confirm:
- Did the applicant show honesty?
- Did answers remain consistent?
- Did the panel note any conduct concerns?
- Does the applicant fit the department culture?
- Would current staff trust this applicant in a live scene?
