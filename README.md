## Recommended Usage Flow

1. Review `docs/interviewer_handbook.md`
2. Run the interview using `interviews/interview_panel_guide.md`
3. Pull questions from the appropriate file in `department_questions/`
4. Score the applicant using `interviews/interview_scorecard.md`
5. Use `docs/hiring_matrix.md` and `docs/probation_recommendations.md`
6. Track status in `docs/applicant_tracking_board.md`
7. Start onboarding with `training/new_hire_onboarding.md`
